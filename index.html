Inclusion Criteria

Listed providers have a first-party MCP server, a provider-managed endpoint, or first-party documentation for deploying MCP servers on their hosting platform. Community-only servers are intentionally left out.

Security Note

MCP tools can operate with the same privileges as the account that authorizes them. Prefer read-only scopes, development projects, and manual approval for tool calls.
