# MCP Hosting Provider Directory

A static GitHub Pages project listing hosting, cloud, and backend providers with official or managed MCP servers.

## Site

The GitHub Pages URL is:

https://mikelavrik.github.io/mcp-hosting-providers/

## Publish on GitHub Pages

This repo can publish from the included GitHub Actions workflow:

1. In repository settings, set Pages to use GitHub Actions if it is not already selected.
2. The workflow in `.github/workflows/pages.yml` publishes the site from the repository root.

The page itself is plain HTML, CSS, and JavaScript. There is no build step.
